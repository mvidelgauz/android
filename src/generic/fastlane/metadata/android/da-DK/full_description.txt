Den selv-hostede produktivitetsplatform som giver dig kontrollen.

Funktioner:
* Let, moderne interface, tilpasset til temaet på din server
* Upload filer til din Nextcloud server
* Del dem med andre
* Hold dine favoritfiler og mapper synkroniserede
* Søg på tværs af alle mapper på din server
* Auto upload for billeder og videoer taget med dit apparat
* Hold dig opdateret med notifikationer
* Multi-konto understøttelse
* Sikker adgang til dine data med fingeraftryk eller PIN
* Integration med DAVx5 (tidligere kendt som DAVdroid) for let opsætning af kalender- og kontaktsynkronisering

Rapporter venligst alle problemer på https://github.com/nextcloud/android/issues og diskuter denne app på https://help.nextcloud.com/c/clients/android

Ny i forhol til Nextcloud? Nextcloud er en privat filsynkroniseringsis- delings- og kommunikationsserver. Det er gratis software, og du kan selv hoste det eller betale et firma for at gøre det for dig. På den måde har du fuldstændig kontrol over dine billeder, din kalender og dine kontaktdata, samt dine dokumenter og alt andet.

Tjek Nextcloud ud på https://nextcloud.com
